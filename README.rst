*Latet anguis in herba* (Virgil, *Eclogues*, 3, 93)

**anguis** is a generic key-store library in Python.

Currently, the following backends are supported:

-  filesystem
-  Etcd
-  Git
-  Redis
-  memcached
-  SFTP
-  Sqlite
-  AWS S3
-  Google Drive

Installation
------------

From source:

::

    $ python setup.py install

From PyPI:

::

    $ pip3 install anguis

Example of usage
----------------

::

    from anguis.fs import AnguisFS

    cache = AnguisFS()

    cache['foo'] = 'bar'
    print(cache['foo'])
    # bar

    del(cache['foo'])
    print(cache['foo'])
    # None
